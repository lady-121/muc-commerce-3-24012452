## 用自己的话回答

- 什么是特征，什么是标签：特征（X）是模型用来做判断的输入信息，比如用户的"使用月数"、"是否投诉"、"消费金额"等数据。标签（y）是我们希望模型预测的答案，在这个任务中就是"Churn"（是否流失）。模型通过学习特征与标签之间的关系，来预测新用户是否会流失。
- 为什么要保留测试集：测试集的作用是评估模型在"没见过的新数据"上的表现。如果在训练模型时让模型看到了测试集的数据，评估结果就会虚高，无法反映模型在真实场景中的泛化能力。所以测试集必须保留，只在模型训练完成后使用一次，用来公正地评估模型效果。
- 为什么83%准确率仍可能没有用：因为数据存在不平衡问题（流失率约16.84%，不流失约83.16%）。一个"永远预测不流失"的模型就能达到83.16%的准确率，但它一个流失用户都找不出来。对于业务来说，找出可能流失的用户并提前干预才是关键，所以只看准确率会误导人，需要结合召回率等指标综合评估。
